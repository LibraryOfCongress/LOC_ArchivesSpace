# LOC ArchivesSpace Public Archive

This repo contains snapshots of plugins developed by the Library of Congress as part of
its ArchivesSpace implementation and customization.

Each time LOC releases updates to its implementation, snapshots will be posted here using
tags in the form `release-YYYY-MM-DD`.
