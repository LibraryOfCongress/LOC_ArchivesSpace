# LOC Database Extension Plugin

## Overview

This is a general plugin for LOC customization to the ArchivesSpace data
model. Most database extensions can be made by creating schema extensions
in the `schema` folder, following the examples in these plugins:

https://github.com/hudmol/donor_details
https://github.com/dartmouth-dltg/local_contexts_projects
https://github.com/hudmol/material_types
